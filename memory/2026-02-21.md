## 白噪音生成器修复（长期问题）
老大反复说白噪音没声音，修了 4 次：
1. 第一次：async/await + resume() — 没用
2. 第二次：全局 click/touch 解锁 + 卡片点击 — 没用
3. 第三次：加了「点击启用音频」全屏引导页 — 老大说没看到按钮（可能缓存）
4. 第四次：**彻底放弃 Web Audio API**，改用 JS 生成 WAV blob + `<audio>` 元素播放 — **终于有声音了！**
5. 但老大说音质差，又改进了：8秒循环+交叉淡入淡出+更自然的音效算法

**教训：Web Audio API 在飞书内置浏览器/某些 WebView 里不可靠。以后涉及音频优先用 `<audio>` 元素 + 生成的 WAV blob。**
