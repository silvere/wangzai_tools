Transform 语法速查表

函数 说明 示例
translate(x, y) 移动元素 translate(50px, 20px)
translateX(x) X 轴移动 translateX(50px)
translateY(y) Y 轴移动 translateY(20px)
rotate(angle) 顺时针旋转 rotate(45deg)
scale(x, y) 缩放元素 scale(1.5, 1.5)
scaleX(x) X 轴缩放 scaleX(-1)
scaleY(y) Y 轴缩放 scaleY(0.5)
skew(x-angle, y-angle) 倾斜变形 skew(10deg, 5deg)
skewX(angle) X 轴倾斜 skewX(10deg)
skewY(angle) Y 轴倾斜 skewY(10deg)
matrix(n,n,n,n,n,n) 组合所有变换 matrix(1,0.5,-0.5,1,0,0)
💡 提示：
  • 多个变换函数可以组合使用，按顺序应用
  • 变换顺序会影响结果：先旋转再平移 和 先平移再旋转结果不同
  • transform-origin 改变变换的中心点位置
  • 变换不影响其他元素布局，只是视觉效果
