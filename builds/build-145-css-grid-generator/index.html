CSS Grid 快速参考

容器属性

  • • display: grid — 启用 Grid
  • • grid-template-columns — 定义列
  • • grid-template-rows — 定义行
  • • gap — 项目间距
  • • justify-content — 水平对齐
  • • align-content — 垂直对齐
  • • grid-template-areas — 命名区域布局

项目属性

  • • grid-column — 列跨度 (start / end)
  • • grid-row — 行跨度 (start / end)
  • • grid-area — 指定区域
  • • justify-self — 单个项目水平对齐
  • • align-self — 单个项目垂直对齐
  • • 快捷写法: grid-column: 1 / -1 跨所有列
  • • repeat(n, 1fr) — n列平均分配
