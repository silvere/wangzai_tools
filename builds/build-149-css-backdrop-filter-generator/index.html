毛玻璃效果

Backdrop-filter 可以让你为一个元素后面区域添加模糊效果，通常用于打造现代UI设计中的半透明毛玻璃卡片。

模糊会穿透到背景，只影响背景，不影响内容文字，非常适合做导航栏、模态框、卡片背景。

💡 提示： 背景使用了随机图片，可以刷新页面换一张背景。毛玻璃效果显示在背景上方，完美还原了真实使用场景。

backdrop-filter 语法速查表

函数 说明 常用值
blur() 对背景应用高斯模糊 4px ~ 20px 常用
brightness() 调整背景亮度 80% ~ 120%
contrast() 调整背景对比度 80% ~ 120%
grayscale() 将背景转为灰度 0% ~ 100%
hue-rotate() 对背景应用色相旋转 0deg ~ 360deg
invert() 反转背景颜色 0% ~ 100%
opacity() 调整背景透明度 0% ~ 100%
saturate() 调整背景饱和度 80% ~ 150%
sepia() 将背景转为棕褐色 0% ~ 100%
none 移除所有滤镜 -
📱 浏览器兼容性：
