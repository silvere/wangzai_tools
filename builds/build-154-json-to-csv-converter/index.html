使用说明: 输入应为 JSON 数组，每个元素是一个对象，嵌套对象会自动展平为 "parent.child" 格式列名

💡 功能特点

  • 支持任意深度嵌套JSON对象自动展平，自动提取所有列
  • 自动检测所有字段作为CSV列，缺失字段留空
  • 支持打开本地JSON文件转换，也支持直接粘贴JSON
  • 支持自定义分隔符（逗号/制表符/分号），支持TSV输出
  • 所有转换完全在浏览器本地完成，文件不会上传到任何服务器
  • 转换完成后可以下载CSV文件或直接复制到剪贴板
  • 特别适合处理评测数据、API返回JSON、导出数据表格等场景
