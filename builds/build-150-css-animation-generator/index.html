CSS Animation 语法速查表

属性 说明 常用值
@keyframes 定义关键帧 @keyframes name { ... }
animation-name 动画名称 any-name
animation-duration 动画持续时间 0s ~ 10s+
animation-timing-function 缓动函数 ease | linear | ease-in | ease-out | cubic-bezier()
animation-iteration-count 重复次数 1 | 3 | infinite
animation-direction 播放方向 normal | reverse | alternate | alternate-reverse
animation-fill-mode 动画结束后样式 none | forwards | backwards | both
transform 变换（在关键帧中使用） translate() rotate() scale()
💡 使用提示：
  • 添加多个关键帧实现复杂动画效果
  • 可以同时给一个元素添加多个动画，用逗号分隔
  • 现代浏览器都支持，不需要前缀
  • forwards 模式会保留动画最后一帧的样式
