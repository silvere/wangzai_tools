border-radius 语法速查

border-radius: 8px; — 四个角都是 8px

border-radius: 8px 16px; — 左上+右下 8px，右上+左下 16px

border-radius: 8px 16px 24px 32px; — 顺时针依次设置左上 右上 右下 左下

border-top-left-radius: 16px; — 只设置左上角

border-radius: 16px / 8px; — 水平半径 16px / 垂直半径 8px（椭圆圆角）

💡 支持百分比 (%)，比如 50% 就可以得到圆形/椭圆形
