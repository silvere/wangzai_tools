权限说明

文件权限

  • r (读): 可以读取文件内容，查看文件信息
  • w (写): 可以修改文件内容，重命名、删除文件
  • x (执行): 可以执行文件（脚本、二进制程序）

目录权限

  • r (读): 可以列出目录下的文件列表
  • w (写): 可以在目录中创建、删除、重命名文件
  • x (执行): 可以进入目录，访问目录下的文件

特殊权限说明

SUID (Set User ID)

文件执行时，进程的有效用户ID变为文件所有者的ID，而非执行者的ID。常用于passwd等需要特殊权限的程序。

SGID (Set Group ID)

文件执行时继承所属组权限；目录中创建的文件会自动继承目录的组权限，常用于共享目录。

Sticky Bit (粘滞位)

设置在目录上时，仅目录所有者、文件所有者和root可以删除或移动目录中的文件，常用于/tmp等公共目录。
